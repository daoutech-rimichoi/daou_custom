# Daoutech Custom Plugin

다우기술 레드마인 환경을 위한 커스텀 플러그인입니다. 초기화면 대시보드 시각화, 일감 상세 페이지 커스텀, 필수 항목 강화 등의 기능을 제공합니다.

## 주요 기능

### 1. 프로젝트별 이슈 현황 대시보드
- 레드마인 접속 시 첫 화면(Welcome index)에 프로젝트별 이슈 처리 현황을 차트(Pie Chart) 형태로 제공합니다.
- 시작일과 종료일을 설정하여 특정 기간 동안 생성된 일감의 현황을 필터링할 수 있습니다. (최대 365일 범위)
- 각 상태별 이슈 개수를 클릭하면 해당 프로젝트의 이슈 목록으로 즉시 이동합니다.

### 2. 일감 상세 페이지 커스텀
- 일감 상세 페이지(`issues/show`) 하단에 다음과 같은 속성들을 추가로 표시합니다:
    - **등록자**: 일감을 생성한 사용자
    - **생성일**: 일감이 생성된 일시
    - **변경일**: 일감이 마지막으로 수정된 일시
    - **종료일**: 일감이 완료 상태로 변경된 일시

### 3. 일감 필수 항목 강화
- 일감 생성 및 수정 시 다음 항목들을 필수값으로 강제합니다:
    - **담당자** (`assigned_to_id`)
    - **범주** (`category_id`)

### 4. 프로젝트 링크 개선
- 프로젝트 이름을 클릭할 때, 프로젝트 정보 페이지가 아닌 해당 프로젝트의 **이슈 목록** 페이지로 바로 이동하도록 링크 동작을 개선했습니다.

### 5. 커스텀 스타일 및 정적 자원
- `Chart.js` (v4.x) 라이브러리를 내장하여 데이터 시각화를 지원합니다.
- 플러그인 전용 CSS를 통해 현대적이고 가독성 높은 대시보드 레이아웃을 제공합니다.

## 환경구성 (테스트 완료 환경)

본 플러그인은 아래 환경에서 개발 및 테스트되었습니다.

- **OS**: Windows (Docker Desktop)
- **Docker Compose**: 2.40.3
- **Redmine**: 6.0.6
- **Ruby**: 3.3.9
- **Puma**: 7.0.1
- **Rails**: 7.2.2.1

## 설치 방법

1. 레드마인의 `plugins` 디렉토리로 이동합니다.
2. 본 저장소를 `daou_custom` 폴더 이름으로 복사(Clone)합니다.
   ```bash
   git clone [repository_url] daou_custom
   ```
3. 레드마인 루트 디렉토리에서 번들 설치를 수행합니다.
   ```bash
   bundle install
   ```
4. 레드마인 서버를 재시작합니다.
